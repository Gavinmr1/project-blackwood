# Development Roadmap

## Overview

Project Blackwood will be developed through incremental milestones.

Each phase should create a playable improvement over the previous phase.

The goal is to validate gameplay before expanding scope.

---

# Phase 0: Pre-Production

Status:
Complete

Goals:
- Define vision
- Create design documents
- Plan architecture

Completed:
- Game Design Document foundation
- Technical planning
- System design

---

# Phase 1: Core Prototype

Goal:
Create the basic ranger experience.

Features:

## Player
- First-person movement
- Looking
- Sprinting
- Crouching
- Interaction

## Environment
- Small test area
- Basic terrain
- Ranger station

## Tools
- Flashlight
- Basic repair tool

## Tasks
- Simple repair task
- Task completion
- Reward system

Success Criteria:
The player enjoys completing basic ranger duties.

---

# Phase 2: Ranger Simulation

Goal:
Create the daily ranger loop.

Features:
- Task generation
- Equipment system
- Inventory
- Ranger vehicle
- Station upgrades

Success Criteria:
A player can complete a full ranger day.

---

# Phase 3: Living Park

Goal:
Make the environment feel alive.

Features:
- Wildlife AI
- Visitor AI
- Weather
- Day/night cycle
- Environmental changes

Success Criteria:
The park feels like a real place.

---

# Phase 4: Cooperative Multiplayer

Goal:
Enable teamwork.

Features:
- Player joining
- Replication
- Shared tasks
- Shared progression

Success Criteria:
Players naturally cooperate.

---

# Phase 5: Mystery Systems

Goal:
Introduce the supernatural.

Features:
- Ranger journals
- Strange events
- Evidence system
- Cryptid behaviors

Success Criteria:
Players create stories about what happened.

---

# Phase 6: Vertical Slice

Goal:
Create a polished demo.

Contains:
- One complete park region
- Ranger station
- Several tasks
- Wildlife
- Multiplayer
- One mystery event

---

# Phase 7: Early Access

Content:
- Additional regions
- More creatures
- More equipment
- More events
- Expanded story

---

# Development Rule

Do not add complexity before the previous system is fun.
